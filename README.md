# 기프티콘 쇼핑몰, Giftishop

Giftishop은 선물하기 편한 기프티콘을 웹사이트 상에서도 구매할 수 있도록 만든 사이트입니다.


## Environment

```
apache-tomcat-9.0.5
java version "9.0.4"
```


## Functions
### 기프티콘 구매

메세지 별로 구분된 리스트에서 당신의 마음을 찾아 선물하세요.

### 장바구니

한꺼번에 구매할 수 있도록 장바구니를 준비했답니다.


### 구매 이력

선물 받은, 구매한 기프티콘을 한 눈에 보세요!
